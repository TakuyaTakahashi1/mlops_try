# Iris モデル評価フロー

このドキュメントでは、Iris デモモデル向けに実装した「評価パイプライン」と「CLI」について説明します。

## 全体像

- 学習済み Iris モデルに対して、sklearn の Iris データセット全体を使って評価を実行
- accuracy / クラス別指標 / 混同行列 を JSON 形式で保存
- 時刻付きファイルと `iris-latest.json` の 2 種類を出力して、履歴と最新値の両方を扱えるようにする
- pytest & CI で評価処理を自動テストし、モデル更新後も同じフローで評価が回るようにしている
- CLI (`python -m ml_sample.eval_cli`) から一発で評価を実行可能

（※中身はあとで一緒に調整してもOK）
